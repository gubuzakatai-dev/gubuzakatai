# Голосовой ввод в первой версии

В первой версии бот не принимает голосовые сообщения и не распознаёт аудио. Владелец диктует запись через микрофон системной клавиатуры телефона; телефон преобразует речь в текст до отправки, а Telegram передаёт боту обычное текстовое сообщение.

Такой текст обрабатывается по общим правилам: одно отправленное сообщение создаёт одну запись; явный временной префикс выбирает список; без префикса запись попадает во «Входящие»; бот присылает текст и место сохранения с кнопкой «Изменить».

Качество и доступность преобразования речи определяются клавиатурой и настройками телефона. Бот не получает аудиофайл, не хранит его, не вызывает сервис распознавания и не может повторить распознавание. Перед отправкой владелец может исправить полученный на телефоне текст.

Если вместо голосового ввода клавиатуры владелец отправит Telegram voice message, бот без звука ответит: «В первой версии отправьте текст. Для диктовки используйте микрофон на клавиатуре телефона». Пользовательская запись из такого сообщения не создаётся.
