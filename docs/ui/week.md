# Экран «Неделя»

## Назначение

Экран показывает задачи, которые владелец намерен выполнить до конца текущей календарной недели, и позволяет отмечать выполнение, отменять отметку, изменять задачу, переносить её или архивировать.

## Границы недели

- Неделя заканчивается в воскресенье включительно по московскому времени.
- Новая неделя начинается в понедельник в 00:00 по Москве.
- Невыполненные задачи не удаляются и не создаются заново при смене недели, а остаются в списке «Неделя».

## Состав списка

- В список входят задачи с текущим временным списком «Неделя».
- Записи других типов здесь не показываются.
- Архивные задачи и записи из скрытой корзины не показываются.
- Выполненные задачи остаются видимыми до ближайшей ежедневной обработки в 00:00 по Москве.

## Порядок

Задачи сортируются по времени добавления в список «Неделя»: самые старые отображаются первыми. Смена недели и перенос задачи не создают копию записи.

## Представление

- Список отображается одним редактируемым служебным сообщением.
- На одной странице показывается не более 10 задач.
- При наличии следующих или предыдущих страниц отображаются кнопки `←` и `→`.
- Каждой задаче соответствует отдельная кнопка-флажок.
- Активная задача отображается с пустым флажком, выполненная — с `✅`.
- Если список пуст, бот показывает сообщение «На эту неделю задач нет» и кнопку «Назад».

## Выполнение

1. Нажатие на флажок активной задачи отмечает её выполненной.
2. Задача остаётся на прежнем месте в списке и получает отметку `✅`.
3. Повторное нажатие до полуночи отменяет выполнение и возвращает активный флажок.
4. В ближайшие 00:00 по Москве выполненная задача скрывается из активного списка независимо от дня недели.

## Действия с задачей

Нажатие на текст задачи открывает меню:

- «Изменить»;
- «Перенести на Сегодня»;
- «Перенести на Завтра»;
- «В архив»;
- «Назад».

Действие применяется к существующей записи и не создаёт дубликат.

## Добавление

Отдельной кнопки `➕ Добавить задачу` на экране нет. Новая задача добавляется обычным текстовым или голосовым сообщением. Бот определяет список по содержимому сообщения либо позволяет исправить место через кнопку «Изменить» в подтверждении.

## Обновление экрана

После выполнения, отмены выполнения, изменения, переноса или архивирования бот редактирует текущее служебное сообщение, сохраняет допустимую страницу и обновляет счётчик на постоянной кнопке «Неделя».

## Ежедневная и недельная обработка

В 00:00 по Москве выполненные задачи скрываются из активного списка. В ночь с воскресенья на понедельник активные задачи остаются в списке «Неделя» без изменения и без создания копий.
