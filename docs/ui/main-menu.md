# Экран «Главное меню»

## Назначение

Главное меню служит точкой прямого перехода к основным разделам SecondBrain и одновременно сохраняет свободный доступ к полю ввода Telegram для новых текстовых записей, в том числе надиктованных через микрофон клавиатуры телефона.

## Открытие меню

- Команда `/start` открывает главное меню и показывает постоянную клавиатуру.
- Отдельная приветственная инструкция не показывается ни при первом, ни при последующих запусках.
- Возврат в главное меню не создаёт текстовую сводку.
- Постоянная клавиатура остаётся доступной внутри разделов, если пользователь не свернул её средствами Telegram.

## Основная навигация

Главная клавиатура содержит кнопки, определённые в `docs/ui/main-screen.md`:

```text
Сегодня (n) | Завтра (n) | Неделя (n)
Поиск       | Папки
```

Нажатие основной кнопки позволяет перейти прямо в соответствующий раздел независимо от текущего открытого подраздела.

## Возврат

- В каждом подразделе есть контекстная кнопка «Назад».
- «Назад» возвращает на один уровень выше, а не обязательно сразу в главное меню.
- На верхнем уровне раздела «Назад» возвращает в главное меню.

## Служебные сообщения

- Для навигации по разделу бот по возможности редактирует текущее служебное сообщение вместо отправки нового.
- Например, экран «Папки» превращается в список разобранных записей, а кнопка «Назад» восстанавливает пункты раздела в том же сообщении.
- Новое сообщение допустимо, когда Telegram технически не позволяет изменить прежнее либо когда бот подтверждает самостоятельное пользовательское действие, например сохранение новой записи.

## Состояние клавиатуры

- После изменения задач или «Входящих» бот обновляет динамические счётчики главной клавиатуры.
- Если пользователь свернул клавиатуру, она снова показывается после `/start` или явного возврата в главное меню.

## Ограничения

Главное меню не показывает приветствие, инструкцию, отдельную сводку, корзину или настройки. Эти возможности не занимают место на основной клавиатуре.
