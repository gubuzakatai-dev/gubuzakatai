# Главный экран SecondBrain

## Формат интерфейса

Главный экран первой версии реализуется средствами Telegram как постоянно доступная клавиатура под полем ввода сообщения. Клавиатуру можно свернуть стандартным элементом Telegram и снова показать при возврате в главное меню.

Отдельное сообщение-сводка при `/start` или возврате в главное меню не отправляется. Текущее состояние отражается непосредственно в подписях кнопок.

## Раскладка

Клавиатура содержит два ряда:

```text
Сегодня (n) | Завтра (n) | Неделя (n)
Поиск       | Папки
```

Отдельной кнопки «Входящие» на главном экране нет. «Входящие» отображаются первым пунктом внутри раздела «Папки».

## Счётчики

- «Сегодня» показывает количество активных задач в списке «Сегодня».
- «Завтра» показывает количество активных задач в списке «Завтра».
- «Неделя» показывает количество активных задач в списке «Неделя».
- «Поиск» не имеет счётчика.
- «Папки» не имеет счётчика; число неразобранных записей показывается рядом с пунктом «Входящие» внутри раздела.
- Если значение счётчика равно нулю, число и скобки не показываются.
- После изменения данных бот обновляет подписи кнопок, чтобы значения оставались актуальными.

## Поведение кнопок

- «Сегодня», «Завтра» и «Неделя» открывают соответствующий временной список задач.
- «Поиск» открывает поиск.
- «Папки» открывает раздел с пунктами «Входящие», «Разобранные» и «Поиск по тегам».

## Приём новых записей

Поле ввода Telegram остаётся основным способом добавления текстовых и голосовых записей. Постоянная клавиатура не должна мешать отправке обычного текста или голосового сообщения.

## Ограничения решения

Главный экран не содержит отдельной текстовой панели, сводки, кнопки «Входящие», настроек или корзины. Доступ к дополнительным действиям проектируется внутри соответствующих разделов.
