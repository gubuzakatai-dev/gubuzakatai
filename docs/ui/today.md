# Экран «Сегодня»

## Назначение

Экран показывает задачи, которые владелец намерен выполнить сегодня, и позволяет быстро отмечать выполнение, отменять отметку, изменять задачу, переносить её или архивировать.

## Состав списка

- В список входят задачи с текущим временным списком «Сегодня».
- Записи других типов здесь не показываются.
- Архивные задачи и записи из скрытой корзины не показываются.
- Выполненные сегодня задачи остаются видимыми до ежедневной обработки в 00:00 по Москве.

## Порядок

Задачи сортируются по времени добавления в список «Сегодня»: самые старые отображаются первыми. Ежедневно перенесённая задача сохраняет исходную дату создания и не становится новой копией.

## Представление

- Список отображается одним редактируемым служебным сообщением.
- На одной странице показывается не более 10 задач.
- При наличии следующих или предыдущих страниц отображаются кнопки `←` и `→`.
- Каждой задаче соответствует отдельная кнопка-флажок.
- Активная задача отображается с пустым флажком, выполненная — с `✅`.
- Если список пуст, бот показывает сообщение «На сегодня задач нет» и кнопку «Назад».

## Выполнение

1. Нажатие на флажок активной задачи отмечает её выполненной.
2. Задача остаётся на прежнем месте в текущем списке и получает отметку `✅`.
3. Повторное нажатие до полуночи отменяет выполнение и возвращает активный флажок.
4. В 00:00 по Москве выполненная задача скрывается из активного списка.

## Действия с задачей

Нажатие на текст задачи открывает меню:

- «Изменить»;
- «Перенести на Завтра»;
- «Перенести на Неделю»;
- «В архив»;
- «Назад».

Действие применяется к существующей записи и не создаёт дубликат.

## Добавление

Отдельной кнопки `➕ Добавить задачу` на экране нет. Новая задача добавляется обычным текстовым или голосовым сообщением. Бот определяет список по содержимому сообщения либо позволяет исправить место через кнопку «Изменить» в подтверждении.

## Обновление экрана

После выполнения, отмены выполнения, изменения, переноса или архивирования бот редактирует текущее служебное сообщение, сохраняет допустимую страницу и обновляет счётчик на постоянной кнопке «Сегодня».

## Ежедневная обработка

В 00:00 по Москве:

1. выполненные задачи скрываются из активного списка;
2. незавершённые задачи остаются в «Сегодня»;
3. активные задачи из «Завтра» переходят в «Сегодня» без создания копий.
