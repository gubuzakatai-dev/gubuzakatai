# Экран «Входящие»

## Назначение

Экран показывает неразобранные записи и позволяет выбрать отдельную запись для ручного разбора или разбора с помощью ChatGPT.

## Открытие

«Входящие» открываются первым пунктом раздела «Папки». Число неразобранных записей отображается на кнопке «Папки» главного меню по правилам `docs/ui/main-screen.md`.

## Состав списка

- Показываются только неразобранные записи во «Входящих».
- Записи в папках, временных списках, архиве и скрытой корзине не показываются.
- Каждая запись существует в единственном экземпляре.
- Записи сортируются по времени поступления: самые старые отображаются первыми.

## Представление

- Список отображается одним редактируемым служебным сообщением.
- Полный текст каждой записи показывается в теле сообщения без сокращения.
- Записи нумеруются, а кнопки с соответствующими номерами позволяют выбрать запись.
- На одной странице показывается не более 10 записей.
- Если полные тексты 10 записей вместе со служебным оформлением не помещаются в лимит сообщения Telegram, страница заканчивается на последней записи, которая помещается целиком. Оставшиеся записи переходят на следующие страницы.
- Одна запись никогда не обрезается и не разделяется между страницами. Если одна запись сама превышает допустимый размер сообщения, она показывается отдельными последовательными служебными сообщениями, но остаётся одной записью и имеет одну кнопку выбора.
- При наличии следующих или предыдущих страниц отображаются кнопки `←` и `→`.
- Если список пуст, бот показывает сообщение «Входящие пусты» и кнопку «Назад».

## Выбор записи

После выбора записи бот показывает её полный текст и кнопки:

- «Разобрать вручную»;
- «Разобрать с ChatGPT»;
- «Назад».

«Назад» возвращает к той странице «Входящих», с которой была открыта запись.

## Способы разбора

- «Разобрать вручную» открывает ручной выбор назначения записи без обращения к ChatGPT.
- «Разобрать с ChatGPT» запускает анализ записи и получение предлагаемой папки.
- Оба способа работают с существующей записью и не создают копию.
- Подробные шаги и действия проектируются в следующем пункте чек-листа — экран «Разбор записи».

## Отсутствие массового действия

Кнопки «Разобрать все» нет. Владелец выбирает и разбирает записи по одной.

## Обновление экрана

После завершения разбора запись исчезает из «Входящих». Бот редактирует список, сохраняет допустимую страницу и обновляет счётчик на кнопке «Папки».
