# Экран «Папки»

## Назначение

Экран объединяет три способа работы с сохранёнными сообщениями: просмотр неразобранных записей, просмотр общего списка разобранных записей и поиск разобранных записей по тегам. Отдельные тематические папки не создаются.

## Открытие и состав

Раздел открывается кнопкой «Папки» главного меню. На кнопке главного меню счётчик не отображается.

Экран содержит пункты в фиксированном порядке:

1. «Входящие (n)»;
2. «Разобранные»;
3. «Поиск по тегам»;
4. «Назад».

`n` — число неразобранных записей во «Входящих». Если записей нет, счётчик рядом с пунктом «Входящие» не показывается. У «Разобранных» и «Поиска по тегам» счётчиков нет.

## Входящие

Пункт «Входящие» открывает экран `docs/ui/inbox.md`. После завершения разбора счётчик этого пункта обновляется.

## Разобранные

- Все разобранные записи находятся в одном общем списке, а не распределяются по тематическим папкам.
- Записи сортируются от новых к старым.
- Показывается полный текст каждой записи, а непосредственно под ним — все назначенные ей теги.
- На странице находится не более 10 записей.
- Если полные тексты не помещаются в лимит сообщения Telegram, число записей на странице автоматически уменьшается; записи не обрезаются.
- При наличии соседних страниц показываются кнопки `←` и `→`.
- Если список пуст, бот показывает «Разобранных записей нет» и кнопку «Назад».

Список отображается одним редактируемым служебным сообщением, когда это допускают ограничения Telegram.

## Действия с разобранной записью

После выбора записи бот показывает полный текст, назначенные теги и кнопки:

- «Изменить текст»;
- «Изменить теги»;
- «Удалить в корзину»;
- «Назад».

«Изменить теги» позволяет назначить одной записи несколько тегов. Изменение тегов не создаёт копий записи. Перед перемещением в скрытую корзину бот запрашивает подтверждение. Отдельного действия и раздела «Архив» нет.

## Поиск по тегам

Пункт открывает отдельный поиск только среди разобранных записей по одному или нескольким тегам. Подробное поведение фильтра проектируется в следующем пункте чек-листа — экран «Поиск».

## Навигация

- «Назад» из «Входящих», «Разобранных» или поиска по тегам возвращает на экран «Папки».
- «Назад» с экрана «Папки» возвращает в главное меню.
