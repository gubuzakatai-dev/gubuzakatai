# Экран «Поиск»

## Назначение

Поиск позволяет найти сохранённую запись по части текста во всех состояниях либо отфильтровать разобранные записи по одному тегу.

## Общий текстовый поиск

Общий поиск открывается отдельной кнопкой «Поиск» главного меню.

1. Бот просит отправить поисковый запрос и показывает кнопку «Назад».
2. Владелец отправляет текст.
3. Бот ищет запрос в отображаемом тексте всех записей и задач.

Поиск охватывает:

- задачи «Сегодня», «Завтра» и «Неделя»;
- неразобранные записи во «Входящих»;
- разобранные записи;
- выполненные записи, скрытые ежедневной обработкой;
- записи в скрытой корзине.

Поиск выполняется без учёта регистра и находит частичное совпадение внутри слова. Например, запрос `поезд` находит текст `Поездка`.

Для сравнения запрос и `display_text` приводятся к Unicode NFC и обрабатываются `casefold`, после чего приложение проверяет вхождение нормализованного запроса в нормализованный текст. Сравнение выполняется в прикладном коде, а не через SQLite `LIKE` или `NOCASE`, которые не гарантируют нужное поведение для кириллицы. Сначала отбираются все совпадения, затем они сортируются и разбиваются на страницы; из-за фильтрации порядок и состав страниц не должны зависеть от регистра текста.

Пустой после удаления окружающих пробелов запрос не запускает поиск: бот повторно просит ввести текст.

## Поиск по тегу

Поиск по тегу открывается пунктом «Поиск по тегам» внутри раздела «Папки».

1. Бот показывает доступные теги и кнопку «Назад».
2. Владелец выбирает ровно один тег.
3. Бот показывает разобранные записи с выбранным тегом.

Одновременно выбрать несколько тегов нельзя. Поиск по тегу не включает «Входящие» и корзину, поскольку он предназначен для активных разобранных записей.

## Результаты

- Результаты сортируются от новых к старым по времени создания исходной записи.
- Полный текст результата показывается без сокращения.
- Под текстом показываются место или состояние записи: «Сегодня», «Завтра», «Неделя», «Входящие», «Разобранные» или «Корзина».
- Если у записи есть теги, они показываются рядом с состоянием.
- На странице находится не более 10 результатов.
- Если полные тексты не помещаются в лимит сообщения Telegram, количество результатов на странице автоматически уменьшается; запись не обрезается.
- При наличии соседних страниц показываются кнопки `←` и `→`.
- Если совпадений нет, бот показывает «Ничего не найдено», кнопки «Новый поиск» и «Назад».

Результаты отображаются одним редактируемым служебным сообщением, когда это допускают ограничения Telegram.

## Действия с результатом

После выбора результата вне корзины бот показывает действия, допустимые для его текущего состояния:

- «Изменить текст»;
- «Изменить теги» только для разобранной нетасковой записи;
- «Сделать задачей» только для разобранной нетасковой записи;
- перенос между временными списками для задачи;
- «В корзину»;
- «Назад».

Для выполненной задачи, уже скрытой полуночной обработкой, вместо обычного переноса показывается «Возобновить». Бот предлагает выбрать «Сегодня», «Завтра» или «Неделя», затем очищает отметку выполнения и скрытия и начинает новый период активности задачи в выбранном списке.

Перед перемещением в корзину требуется подтверждение.

«Сделать задачей» предлагает выбрать «Сегодня», «Завтра» или «Неделя». После подтверждения все теги снимаются, а существующая запись становится задачей выбранного списка без создания копии.

Для результата из корзины доступны только:

- «Восстановить»;
- «Назад».

Восстановление возвращает существующую запись в состояние и временной список, которые она имела непосредственно перед перемещением в корзину. Копия не создаётся.

Отдельного экрана или команды просмотра всей корзины нет. Общий текстовый поиск — единственный пользовательский маршрут к находящейся в корзине записи и её восстановлению.

## Повторный поиск и возврат

- «Новый поиск» возвращает к вводу текста в общем поиске или к выбору тега в поиске по тегу.
- «Назад» из карточки результата возвращает на прежнюю страницу результатов.
- «Назад» из общего поиска возвращает в главное меню.
- «Назад» из поиска по тегу возвращает на экран «Папки».
