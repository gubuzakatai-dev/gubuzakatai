# Экран «Завтра»

## Назначение

Экран показывает задачи, которые владелец намерен выполнить завтра, и позволяет заранее отметить выполнение, отменить отметку, изменить задачу, перенести её или архивировать.

## Состав списка

- В список входят задачи с текущим временным списком «Завтра».
- Записи других типов здесь не показываются.
- Архивные задачи и записи из скрытой корзины не показываются.
- Выполненные задачи остаются видимыми до ежедневной обработки в 00:00 по Москве.

## Порядок

Задачи сортируются по времени добавления в список «Завтра»: самые старые отображаются первыми. Перенос изменяет временной список существующей записи и не создаёт копию.

## Представление

- Список отображается одним редактируемым служебным сообщением.
- На одной странице показывается не более 10 задач.
- При наличии следующих или предыдущих страниц отображаются кнопки `←` и `→`.
- Каждой задаче соответствует отдельная кнопка-флажок.
- Активная задача отображается с пустым флажком, выполненная — с `✅`.
- Если список пуст, бот показывает сообщение «На завтра задач нет» и кнопку «Назад».

## Выполнение

1. Нажатие на флажок активной задачи отмечает её выполненной.
2. Задача остаётся на прежнем месте в списке и получает отметку `✅`.
3. Повторное нажатие до полуночи отменяет выполнение и возвращает активный флажок.
4. В 00:00 по Москве выполненная задача скрывается из активного списка и не переносится в «Сегодня».

## Действия с задачей

Нажатие на текст задачи открывает меню:

- «Изменить»;
- «Перенести на Сегодня»;
- «Перенести на Неделю»;
- «В архив»;
- «Назад».

Действие применяется к существующей записи и не создаёт дубликат.

## Добавление

Отдельной кнопки `➕ Добавить задачу` на экране нет. Новая задача добавляется обычным текстовым или голосовым сообщением. Бот определяет список по содержимому сообщения либо позволяет исправить место через кнопку «Изменить» в подтверждении.

## Обновление экрана

После выполнения, отмены выполнения, изменения, переноса или архивирования бот редактирует текущее служебное сообщение, сохраняет допустимую страницу и обновляет счётчик на постоянной кнопке «Завтра».

## Ежедневная обработка

В 00:00 по Москве:

1. выполненные задачи скрываются из активного списка «Завтра» и не переносятся;
2. активные задачи переходят в «Сегодня» без создания копий;
3. после обработки список «Завтра» пуст, пока в него не будет добавлена или перенесена новая задача.
